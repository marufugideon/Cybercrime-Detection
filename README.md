## Overview
The Cyber Crime Prediction System is a machine learning project developed to analyse and predict cyber crime patterns using historical data. The system helps identify possible cyber threats and supports data-driven security analysis.

## Features
- Analyze cyber crime datasets
- Predict cyber crime trends
- Data preprocessing and visualization
- Machine learning model prediction
- User-friendly interface

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- RandomForest
- Matplotlib
- Flask

## Installation

1. Clone the repository
git clone https://github.com/marufugideon/Cybercrime-Detection.git

2. Open the project folder
cd Cybercrime-Detection

3. Create virtual environment
python -m venv venv

4. Activate environment

### Windows
venv\Scripts\activate

5. Install dependencies
pip install -r requirements.txt

6. Run the project
python app.py
